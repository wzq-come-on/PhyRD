# Legacy material

Nothing under this directory is part of the active PhyRD package or training
entry points. It contains recoverable pre-refactor archives only.

The maintained code paths are:

- `src/phyrd/models/deterministic/` for deterministic backbone adapters;
- `third_party/` for pinned upstream sources;
- `scripts/train.py` for training;
- `configs/` for experiment configuration;
- `artifacts/` for run outputs.
